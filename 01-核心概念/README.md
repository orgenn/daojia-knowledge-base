# 核心概念模块导航

本模块用于承接道家与道教相关的基础知识、修持方法、术数应用、文化理解与常见问答，是整个知识库的概念底座。

## 建议阅读顺序

1. [[01-思想与文化/道家哲学]]：先理解道家思想的核心框架。
2. [[01-思想与文化/道家经典导读]]：再进入经典与现代转译。
3. [[02-道教体系/道教历史与宗派]]：区分道家思想与道教体系。
4. [[03-修持与养生/内丹修炼体系v2]]：理解修持、养生与实践边界。
5. [[04-术数与应用/八字命理]]、[[04-术数与应用/风水学]]、[[04-术数与应用/奇门遁甲]]：进入术数应用。
6. [[05-仪式与符箓/符箓体系]]：用文化与合规视角理解仪式内容。
7. [[06-案例与答疑/道家实修问答]]：回到答疑与现代使用场景。

## 目录结构

- [[01-思想与文化/README]]：思想根基、经典导读、文化表达
- [[02-道教体系/README]]：历史、宗派、节日、神仙观相关条目
- [[03-修持与养生/README]]：内丹、导引、饮食、戒忌、日常养护
- [[04-术数与应用/README]]：八字、奇门、风水、相术等应用知识
- [[05-仪式与符箓/README]]：仪式、符箓、物品与相关文化说明
- [[06-案例与答疑/README]]：实战案例、入门答疑、常见误区

## 使用原则

- 优先区分“思想体系”“宗教体系”“实践方法”“术数应用”四个层级。
- 涉及仪式、符箓、禁忌时，统一采用文化说明与合规表达，不直接使用高风险营销口径。
- 同主题存在多版本时，以内容更完整、结构更清晰的版本为主；旧版本保留作对照参考。
