## 一、什么是八字

**八字**又称四柱预测学，通过出生时间（年、月、日、时）天干地支的组合来分析人生规律。

### 基础概念

| 概念 | 说明 |
|------|------|
| 四柱 | 年柱、月柱、日柱、时柱 |
| 天干 | 甲乙丙丁戊己庚辛壬癸（10个） |
| 地支 | 子丑寅卯辰巳午未申酉戌亥（12个） |
| 五行 | 金木水火土 |
| 阴阳 | 天干地支分阴阳 |

## 三、格局系统

### 格局定义

格局是八字分析的整体框架，反映人生主要趋向。

### 主要格局

| 格局 | 特征 | 适合职业 |
|------|------|----------|
| 官杀格 | 权力管理 | 公务员、管理层 |
| 财格 | 财富追求 | 商业、金融 |
| 印绶 | 学术研究 | 教育、科研 |
| 食伤格 | 才华创作 | 艺术、演艺 |
| 禄刃格 | 创业拼搏 | 创业、自由职业 |
| 从格 | 特殊命局 | 特殊机遇 |

### 取格方法

1. **月令取格**：月支为本气透干
2. **用神辅助**：配合用神判断高低
3. **综合分析**：结构平衡
