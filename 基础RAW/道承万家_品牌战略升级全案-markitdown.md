# **第一部分 · 整体规划提升建议**

基于四份核心规划文档的深度分析，以下提出九个方向的战略级提升建议。

## **1.1 品牌定位升级——从"平台"到"运动"**

*现有定位：「中国首个以道家五术为核心的家庭能量管理平台」*

提升方向：平台是工具，运动是信仰。建议同步推出"百岁中国·家庭能量运动"作为社会化叙事框架，让用户成为运动参与者而非产品消费者。

|  |  |
| --- | --- |
| **当前表达** | **升级建议** |
| 家庭能量管理平台 | 百岁中国 · 家庭能量运动发起者 |
| 传统文化现代传播 | 重建中国家庭的生命质量标准 |
| 道家五术服务 | 让千年智慧成为现代家庭的操作系统 |
| 知识付费产品 | 生命成长的陪伴型社群 |

## **1.2 IP战略——从"品牌"到"人"**

当前规划以机构品牌为主，缺少强个人IP支撑。4A级品牌必须有人格化的核心IP。建议：

* 主理人IP：以"文化行者"人设出圈，不是专家，是同行者
* IP核心标签：80后 · 真修行者 · 两个孩子的父/母 · 用道家方法解决现代焦虑
* 内容核心句式："我用这个方法，解决了\_\_\_，你也可以"
* 差异点：不说教，只分享；不神秘，只真实；不预测，只陪伴

## **1.3 内容战略——从"发布"到"造势"**

阶段1内容规划为每周2-5条，属于常规运营节奏，缺少"造势"动作。建议增加以下内容战略：

* 发起一个挑战：#百岁无忧计划# 打卡挑战，21天养生习惯
* 制造一个话题：每季度发布《中国家庭能量报告》，制造媒体传播节点
* 建立一个仪式：每年冬至日"百岁雅集"，成为品牌年度高光时刻
* 打造一个标志性内容：将五术五句话做成经典金句卡片，全网传播

## **1.4 私域战略——从"社群"到"家族"**

建议将私域社群升级为"道承家族"体系，建立更强的归属感和身份认同：

## **1.5 产品战略——增加"体验锚点"**

## **3.3 内容创作规范**

**内容调性量表**

内容创作时，对照以下量表自检：

|  |  |  |  |
| --- | --- | --- | --- |

**禁止内容类型**

* 涉及具体疾病治疗的内容
* 对具体个人或企业做命运判断的内容
* 宣称可以预测未来事件（股票、婚姻、事业结果）
* 贬低其他传统文化品牌或从业者
* 使用恐吓或制造焦虑的营销手法（如"不改运就会..."）
* 无依据的医学或科学宣称
