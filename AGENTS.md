# AGENTS

> 本仓库是以智能体协作为核心的道家文化 Wiki 知识库，承载徐州云笈文化项目。

## 启动顺序（必读）

1. `08-LLM-Wiki知识管理/AGENT接入必读.md` — 工作闭环与禁止事项
2. `08-LLM-Wiki知识管理/全库入库规则.md` — 模块归属与入库检查
3. `04-合规规范/运营规则总纲.md` — **内容输出前必须过一遍**
4. `05-项目资料/公司信息.md` — 项目主体与品牌定位
5. `08-LLM-Wiki知识管理/README.md` — 顶层治理入口

## 工作原则

| 原则 | 说明 |
|------|------|
| 合规优先 | 营销/课程材料输出前必须过合规框架，禁止词汇见`运营规则总纲` |
| 归档明确 | 新内容必须归属到 01-核心概念 / 02-商业运营 / 03-教育内容 / 04-合规规范 / 05-项目资料 / 06-角色设定 / 07-经典书籍 |
| 不重复 | 禁止把同一内容复制到多个模块 |
| 补链接 | 新增内容必须挂入口链接到模块根 README |

## 关键禁忌

- 禁止输出：算命、占卜、改运、治疗、治愈、效果承诺
- 禁止：自称道士/大师、做法事、画符、承诺改命
- 平台敏感词：「空亡」「克父/克母/克夫/克妻」「小人」「桃花劫」
- 合规转译：「吉凶」→「能量状态」/「适合/不适合」；「算命」→「性格分析」/「天赋解读」
