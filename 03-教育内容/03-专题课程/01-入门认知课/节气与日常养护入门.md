## 课程核心理念

### 为什么要关注节气？

现代人最大的健康问题：**跟天对着干**。

- 夏天狂吹空调，该出汗的时候不出
- 冬天夜夜烧烤，该藏的时候不藏
- 一年四季熬夜，身体不知道什么时候该休

古人说："人与天地相参，与日月相应。"
意思是：人也是自然的一部分，跟着自然的节奏走，身体会舒服；跟自然对着干，身体会乱。

### 节气养护不是玄学

节气养护的本质是：
- 跟着太阳的节奏起居
- 跟着四季的变化调节饮食
- 跟着气候的特点调整活动

这不是迷信，是**顺应生理规律**。

- 情绪：保持平和，避免急躁

### 饮食调理原则

**春季**：少酸多甘，养肝护脾
- 推荐：韭菜、菠菜、芹菜、山药、大枣
- 避免：过多酸味食物（如酸梅、柠檬）

**夏季**：清淡为主，清热解暑
- 推荐：苦瓜、黄瓜、绿豆、西瓜、冬瓜
- 避免：过多油腻、辛辣食物

**秋季**：滋阴润燥，养肺为主
- 推荐：梨、百合、银耳、山药、蜂蜜
- 避免：过多辛辣、油炸食物

**冬季**：温补为主，护阳养肾
- 推荐：羊肉、牛肉、韭菜、核桃、黑芝麻
- 避免：生冷食物、过度饮酒

### 运动建议

| 季节 | 运动类型 | 运动强度 | 注意事项 |
|------|----------|----------|----------|
| 春季 | 户外活动、舒展运动 | 中等强度 | 多晒太阳，活动筋骨 |
| 夏季 | 清晨或傍晚运动、游泳 | 轻度到中等 | 避免暴晒，及时补水 |
| 秋季 | 登山、徒步、太极 | 中等强度 | 适度出汗，避免过劳 |
| 冬季 | 室内运动、瑜伽、太极 | 轻度运动 | 保暖为主，避免大汗 |

## 常见养生误区

### 误区一：春夏不保暖

**错误观点**：春夏天气热，不需要保暖
**正确理解**：
- 春季乍暖还寒，容易感冒，要注意"春捂"
- 夏季空调不要温度太低，避免寒邪入侵

### 误区二：秋冬过度进补

**错误观点**：秋冬就是大补特补的时候
**正确理解**：
- 进补要量力而行，不要过度
- 体虚者进补，体实者不需要
- 过度进补反而会上火、消化不良

### 误区三：冬天完全不动

**错误观点**：冬天太冷，应该完全休息
**正确理解**：
- 冬季要"藏"，但不代表不动
- 适度活动有助于气血运行
- 避免大汗淋漓，要"微汗即止"

### 误区四：只喝热水

**错误观点**：多喝热水总没错
**正确理解**：
- 夏季可以喝温开水，不要太烫
- 也可以喝凉白开，但不喝冰水
- 根据体质和个人感受调整水温

## 21天节气实践打卡计划

### 第一周（7天）：建立感知

**目标**：开始观察自己的身体反应

**每日任务**：
1. 记录当天的节气和天气
2. 观察身体感觉（是否容易疲劳、睡眠质量等）
3. 按照当前季节的起居建议调整作息
4. 记录饮食变化

**打卡格式**：
```
日期：XX月XX日
节气：XX
天气：晴/阴/雨
身体感觉：良好/一般/疲劳
睡眠质量：好/中/差
今日饮食特点：
今日执行建议：
```

### 第二周（7天）：调整行为

**目标**：开始按照节气建议调整生活习惯

**每日任务**：
1. 按照季节建议调整起床和就寝时间
2. 选择2-3种当季推荐食物加入日常饮食
3. 选择适合当前季节的运动类型
4. 记录身体感受变化

**打卡格式**：
```
日期：XX月XX日
今日早起：是/否
今日就寝：是/否
今日推荐食物：[列出]
今日运动：[运动类型+时长]
身体感受变化：
```

### 第三周（7天）：形成习惯

**目标**：让节气养生成为生活习惯

**每日任务**：
1. 复盘前两周的实践效果
2. 找出最适合自己身体的2-3条建议
3. 制定长期执行计划
4. 分享实践心得

**打卡格式**：
```
日期：XX月XX日
复盘总结：[哪些建议对自己有效]
长期计划：[决定长期坚持的3条建议]
心得分享：
```
