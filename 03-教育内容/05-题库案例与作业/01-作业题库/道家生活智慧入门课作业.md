> 本作业题库用于检验学习效果，巩固知识点，引导实践应用

---

## 作业说明

**作业目的**：
- 检验知识理解度
- 引导实践应用
- 建立学习习惯

**完成要求**：
- 每日作业约需10-15分钟
- 建议当日完成，不要积压
- 真诚记录，方便复盘

**提交方式**：
- 可在社群打卡分享
- 可私信班主任

## 第1天作业：什么是"活得明白"？

### 基础题

1. **选择题**：道家认为，人焦虑的根本原因是：
   - A. 运气不好
   - B. 心乱了，想要的太多
   - C. 外界环境不好
   - D. 能力不够

2. **判断题**：道家智慧认为，只要努力就会成功，没有其他因素。
   - 正确（ ）
   - 错误（ ）

### 实践题

3. **焦虑清单**：写下今天让你感到焦虑的三件事，并标记哪些是你能控制的，哪些不能。

| 序号 | 让我焦虑的事 | 能否控制 | 分析 |
|------|-------------|---------|------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**思考引导**：
- 哪些事情其实你无法改变？
- 对于无法改变的事，焦虑有用吗？
- 你能把注意力转移到能控制的事情上吗？

### 思考题

4. **自由回答**：你觉得什么叫"活得明白"？现在你的回答和课程开始前有什么不同？
