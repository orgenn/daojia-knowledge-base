# 题库案例与作业

本目录用于承接练习题、案例拆解、课后作业、答疑清单与学员反馈。

---

## 目录结构

### 01-作业题库
用于收集和整理各课程的作业题目、练习题、自测题

### 02-经典案例
用于整理经典案例拆解、实例分析、应用场景

### 03-常见问题答疑
用于收集学员常见问题、标准答案、补充说明

### 04-学习效果测评
用于设计学习前后的测评工具、进步跟踪

---

## 使用原则

- 题目设计要符合合规要求，避免敏感词汇
- 案例要具有代表性和启发性
- 答疑要保持客观中性的态度
- 所有内容要标注课程来源和使用场景
