## 课程层级结构

```
第一层：入门认知课（零门槛，快速体验）
├── 道家生活智慧入门课
├── 节气与日常养护入门
└── 家居秩序与环境观察入门

第二层：综合基础课（建立框架，兴趣筛选）
├── 八字与风水公开课
└── 奇门面相风水初级班

第三层：专题系统课（深度学习，专业能力）
├── 奇门遁甲系统课
│   └── 静幻奇门基础课
├── 八字系统课
│   └── 八字命理系统基础
└── 风水系统课
    └── 风水环境学基础

第四层：交付与进阶课（实践深化）
├── 主题训练营
│   └── 道家生活智慧践行营
├── 线下沙龙/工作坊
│   └── 道家智慧线下沙龙
└── 企业文化工作坊
```

## 课程关联关系

### 入门认知课关联

**道家生活智慧入门课**
- 向上关联：无（最基础课程）
- 向下关联：节气与日常养护入门、综合基础课系列
- 横向关联：所有入门认知课
- 学习目标：建立道家生活智慧的基本认知
- 适合人群：对传统文化感兴趣的大众，寻求内心平静的都市人

**节气与日常养护入门**
- 向上关联：道家生活智慧入门课
- 向下关联：综合基础课系列
- 横向关联：家居秩序与环境观察入门
- 学习目标：建立顺应自然节律的生活方式
- 适合人群：希望改善亚健康状态的人群

**家居秩序与环境观察入门**
- 向上关联：道家生活智慧入门课
- 向下关联：风水系统课
- 横向关联：节气与日常养护入门
- 学习目标：改善居家环境，提升生活品质
- 适合人群：对空间美学感兴趣的人群

### 综合基础课关联

**八字与风水公开课**
- 向上关联：入门认知课系列
- 向下关联：八字系统课、风水系统课
- 横向关联：奇门面相风水初级班
- 学习目标：术数启蒙与兴趣筛选
- 适合人群：对传统术数有初步兴趣的学员

**奇门面相风水初级班**
- 向上关联：入门认知课系列
- 向下关联：奇门遁甲系统课、面相专题课
- 横向关联：八字与风水公开课
- 学习目标：建立基础术数理解框架
- 适合人群：对面相、风水、奇门感兴趣的初学者

### 专题系统课关联

**静幻奇门基础课**
- 向上关联：奇门面相风水初级班
- 向下关联：高级奇门课程、实战应用课程
- 横向关联：八字系统课、风水系统课
- 学习目标：构建奇门完整知识与应用框架
- 适合人群：希望系统学习奇门遁甲的学员

**八字命理系统基础**
- 向上关联：八字与风水公开课
- 向下关联：高级命命理课程
- 横向关联：奇门遁甲系统课、风水系统课
- 学习目标：构建命理基础与案例分析能力
- 适合人群：对命理学感兴趣的学员

**风水环境学基础**
- 向上关联：八字与风水公开课、家居秩序与环境观察入门
- 向下关联：高级风水课程
- 横向关联：奇门遁甲系统课、八字系统课
- 学习目标：构建空间观察与调整能力
- 适合人群：对风水环境学感兴趣的学员

### 交付与进阶课关联

**道家生活智慧践行营**
- 向上关联：道家生活智慧入门课
- 向下关联：进阶践行营
- 横向关联：线下沙龙
- 学习目标：把学到的知识真正变成生活习惯
- 适合人群：希望提升完课率、持续践行的学员

**道家智慧线下沙龙**
- 向上关联：所有课程
- 向下关联：其他线下活动
- 横向关联：主题训练营
- 学习目标：建立高信任体验和深度连接
- 适合人群：希望深度学习和真实连接的学员

## 学习路径建议

### 路径一：道家生活智慧路径

```
道家生活智慧入门课 → 节气与日常养护入门 → 家居秩序与环境观察入门 → 道家生活智慧践行营
```

**适合人群**：希望改善生活质量、建立内心平静的大众
**学习重点**：生活智慧、日常养护、环境改善、持续实践

### 路径二：术数专业学习路径

```
八字与风水公开课 → 奇门面相风水初级班 → 静幻奇门基础课 → 高级奇门课程
```

**适合人群**：希望系统学习传统术数的学员
**学习重点**：奇门遁甲、面相学、风水学

### 路径三：系统学习路径

```
入门认知课 → 八字与风水公开课 → 选择专题系统课 → 主题训练营或线下沙龙
```

**适合人群**：希望全面了解和发展的学员
**学习重点**：先建立生活基础，再选择专业方向，最后深化实践
