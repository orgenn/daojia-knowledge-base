# Channel Playbooks

## 抖音

### 目标

- 抢注意力
- 打开认知
- 导向私域或直播

### 适合内容

- 反常识观点
- 生活习惯问题
- 节气养生提醒
- 家庭/职场/空间的小方法

### 推荐结构

1. 强钩子
2. 一个问题
3. 一个方法
4. 一个 CTA

## 小红书

### 目标

- 搜索曝光
- 信任建立
- 私信与收藏

### 适合内容

- 案例拆解
- 家庭秩序与情绪稳定
- 养生生活方式清单
- 节气内容、空间整理、美学表达

### 推荐结构

1. 封面强标题
2. 用户痛点
3. 方法分点
4. 收藏/私信 CTA

## 视频号

### 目标

- 深度信任
- 直播承接
- 微信生态转化

### 适合内容

- 主理人观点
- 直播答疑
- 活动记录
- 社群与课程成果片段

### 推荐结构

1. 观点开场
2. 真实问题
3. 方法和案例
4. 直播或社群 CTA

## 私域

### 目标

- 培育
- 成交
- 复购

### 内容节奏

- 每天：轻触达
- 每周：主题内容 + 直播/活动
- 每月：转化活动 + 案例复盘

## 绝对避免

- 算命、改运、法事、开光
- 治疗、治愈、包准、逆天改命
- 恐吓式标题和焦虑式转化
