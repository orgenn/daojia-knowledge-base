# Project Context

## 项目目标

徐州云笈文化是基于现有“道承万家”知识库衍生出的徐州本地化落地版本，方向为：

- 传统文化传播
- 现代生活方式内容
- 家庭、养生、关系、职场与空间相关知识服务
- 低风险、合规、可持续运营

## 关键原则

- 不做迷信承诺
- 不做医疗承诺
- 不替用户做重大决策
- 所有内容都要能转译成现代生活表达

## 高价值研究主题

- 徐州本地适合合作的文化场地与活动空间
- 徐州及周边家庭教育、国学、养生、文旅活动供给
- 用户对“传统文化生活方式化”的接受方式
- 本地内容与活动如何避开高风险口径
- 哪些本地资源可与内容、社群、活动联动
