# AI合规检查接口

> 用于智能体内容检查与转译的简化接口

---

## 📋 接口说明

本接口定义了AI系统检查内容的规则和转译逻辑，供智能体调用。

---

## 🔍 接口一：内容风险检查

### 功能
检查文本内容是否包含违规词汇和表达

### 参数
- `input_text`: 待检查的文本

### 返回
```json
{
  "risk_level": "🔴 高风险" | "🟡 中风险" | "🟢 低风险",
  "flagged_terms": ["违规词1", "违规词2"],
  "suggestions": ["建议1", "建议2"],
  "needs_disclaimer": true | false
}
```

### 规则
**高风险**：
- 包含医疗承诺词汇
- 包含改运/法事收费相关
- 包含重大决策承诺（发财、改命）

**中风险**：
- 包含平台敏感词（空亡、克父等）
- 缺少免责声明
- 包含轻微违规词汇

**低风险**：
- 已进行合规转译
- 包含完整免责声明
- 未检测到风险内容

## ✅ 接口三：免责声明检查

### 功能
检查文本是否包含必要的免责声明

### 参数
- `input_text`: 待检查的文本

### 返回
```json
{
  "has_disclaimer": true | false,
  "disclaimer_type": "完整" | "部分" | "缺失",
  "recommended_disclaimer": "推荐的免责声明文本"
}
```

### 推荐的免责声明模板

```javascript
const DISCLAIMER_TEMPLATES = {
  // 通用版
  general: "本内容基于中华传统文化研究，仅供参考，不构成任何决策依据。请理性看待。",

  // 直播版
  live: "本直播内容仅供文化交流与参考，不构成投资、择业、婚恋、医疗等决策依据。每个人的人生掌握在自己手中。",

  // 短视频版
  video: "以上内容基于传统文化视角，仅供学习参考，请理性看待，勿迷信盲从。",

  // 医疗相关
  medical: "本内容仅供参考，不构成医疗建议。如有身体不适，请咨询专业医师。"
}
```

## 📝 接口四：话术模板推荐

### 功能
根据内容类型推荐合适的话术模板

### 参数
- `content_type`: 内容类型 ("live", "video", "article", "consultation")

### 返回

  "key_points": ["要点1", "要点2"],
  "placeholders": ["可替换字段1", "可替换字段2"]

## 🤖 使用示例

### 示例1：检查文本风险

```python
# 伪代码示例
input_text = "今天用奇门遁甲给大家算一卦，看看你什么时候发财"

result = check_risk(input_text)

# 返回结果
{
  "risk_level": "🔴 高风险",
  "flagged_terms": ["算一卦", "发财"],
  "suggestions": [
    "将'算一卦'改为'分析当下能量'",
    "将'发财'改为'提升财富敏感度'",
    "添加免责声明"
  ]
}
```

### 示例2：转译文本

```python
input_text = "你命中克父，需要做法事化解"

result = translate_text(input_text)

# 返回结果
{
  "original_text": "你命中克父，需要做法事化解",
  "translated_text": "你与父亲缘分特别，建议调整状态",
  "changes": [
    {"original": "命中克父", "translated": "与父亲缘分特别"},
    {"original": "做法事化解", "translated": "调整状态"}
  ]
}
```
