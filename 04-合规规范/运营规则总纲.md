## 🏛️ 1.5 国家政策支持与禁止

### 国家支持什么？（项目合法依据）

根据中共中央办公厅、国务院办公厅印发的《关于实施中华优秀传统文化传承发展工程的意见》，国家明确支持：

| 支持方向 | 具体内容 |
|---------|---------|
| 核心思想理念 | 道法自然、天人合一、革故鼎新、与时俱进等 |
| 中华传统美德 | 自强不息、敬业乐群、孝老爱亲等 |
| 人文精神 | 和而不同、俭约自守、中和泰和等 |
| 融入生产生活 | 传统养生、中华美食、传统节日等 |

**项目定位**：
- "道家养生、亲子八段锦、家庭和谐、品德养成"——完全属于国家支持的范畴
- 国家还在搞"文化进万家"活动，鼓励优质文化资源直达基层

### 国家禁止什么？（红线）

**严厉打击的是"借教敛财"和"封建迷信活动"**

| 禁止行为 | 政策依据 | 后果 |
|---------|---------|------|
| 以宗教名义进行商业宣传 | 《宗教事务条例》明确禁止 | 没收所得+罚款 |
| 假借道教名义开展活动、谋取利益 | 12部门联合文件 | 治安拘留+罚款 |
| 算命、看事儿、风水改运等封建迷信活动 | 地方专项整治行动 | 店面整改+行政处罚 |
| 假冒道教教职人员招摇撞骗 | 宗教事务条例+治安管理处罚法 | 拘留+罚款 |

**典型案例**：
- 法库县2025年专项整治中，对张贴"看事儿""算卦""立堂口"等标语、非宗教教职人员穿道袍从事经营活动的15家店铺进行了清理整治。

### 核心红线

**绝对禁止**：
- ❌ 不能自称"道士""道长"
- ❌ 不能做"法事""画符""驱邪"
- ❌ 不能承诺"改运""消灾"
- ❌ 不能在道观内搞商业活动
- ❌ 不能用道教名义进行商业宣传

### 道教活动 vs 文化活动（关键区分）

| 维度 | 道教宗教活动 | 你的文化活动 |
|------|-------------|-------------|
| 主体 | 道教教职人员、道教活动场所 | 文化传播公司、咨询师 |
| 场所 | 依法登记的道观 | 书院、民宿、文化空间 |
| 内容 | 皈依、传度、科仪、法事 | 养生功法、环境咨询、静心体验 |
| 目的 | 宗教信仰 | 身心健康、文化传承 |
| 资质 | 宗教活动场所登记证 | 营业执照、相关咨询资质 |

**一句话**：你在道观外、穿常服、讲养生，没问题。

### 实操建议

| 要做（绿灯） | 不要做（红灯） |
|-------------|-------------|
| ✅ 注册文化传播/健康管理类公司 | ❌ 注册宗教类机构 |
| ✅ 穿汉服、禅服、常服 | ❌ 穿道袍、法衣 |
| ✅ 讲养生、讲环境美学、讲天赋分析 | ❌ 讲风水改运、算命、法事 |
| ✅ 在书院、民宿、文化空间办活动 | ❌ 在道观内办收费活动 |
| ✅ 按"咨询服务"收费 | ❌ 按"功德""供养"收费 |
| ✅ 宣传"身心健康""家庭和谐" | ❌ 宣传"消灾""改运""超度" |

### 总结

国家政策的态度很清晰：
- 支持道家文化作为优秀传统文化传播
- 打击借道教名义的商业敛财和封建迷信活动

"道承万家"项目，定位是"文化传播"和"健康管理"，做的是养生功法、家庭咨询、亲子活动——这在国家政策上完全站得住脚。

只要不越红线（不做法事、不改运承诺、不穿道袍、不在道观搞商业），这就是一个阳光下的合法生意。

- [ ] 不包含"空亡、克父、克母、克夫、克妻"等平台敏感词
- [ ] 不包含"包准、包灵、保证"等承诺词汇
- [ ] 不包含"血光之灾、灾难、破财"等恐吓表述

**表达检查**：
- [ ] 不做出具体承诺（如"你能赚100万"）
- [ ] 不暗示或明示具备预测能力
- [ ] 不做出绝对化的判断（如"注定"、"一定"）
- [ ] 使用"建议"、"适合"、"参考"等中性词汇

**合规声明**：
- [ ] 添加必要的免责声明
- [ ] 标明内容为"传统文化研究"或"文化视角"
- [ ] 提醒用户"理性看待"、"自主决策"

**平台特定检查**：
- [ ] 小红书：简介、正文、评论区不包含微信号/外链
- [ ] 抖音：不使用平台明令禁止的玄学词汇
- [ ] 视频号：遵守腾讯平台的审核规则

### 4.2 直播过程检查

**开场检查**：
- [ ] 开场白包含免责声明
- [ ] 明确定位为"传统文化研究者"而非"算命大师"
- [ ] 说明内容为"文化参考"而非"预测"

**互动检查**：
- [ ] 不替观众做重大决策（医疗、婚姻、官司、投资）
- [ ] 不承诺具体结果（发财、改命、治愈）
- [ ] 对敏感问题使用合规转译回应

**常见问题合规回答**：

| 用户提问 | 违规回答 | 合规回答 |
|---------|---------|---------|

| "我和我老公会不会离婚？" | "你婚姻宫不好，会离" | "我不做这种预测。但我可以帮你看，你们俩的能量类型是不是匹配，沟通方式怎么调会更顺。" |
| "我该不该跳槽？" | "跳，下个月有贵人" | "我可以帮你看当下的能量适不适合做变动。但最终决定权在你。我的角色是帮你看到你可能忽略的维度。" |

### 4.3 课程交付检查

**课程宣传检查**：
- [ ] 不夸大课程效果（"包你学会"、"保证月入过万"）

**教学内容检查**：

- [ ] 不涉"封建迷信"、"替代医疗/法律/理财决策"
- [ ] 使用合规转译的话术体系

## 🛡️ 五、三条运营铁律（核心原则）

### 铁律一：不说鬼神——只说环境、心理、文化、养生

**禁止说**：
- ❌ 鬼神、灵异、附体、开光、做法
- ❌ 阴阳、驱邪、超度、还阴债

**可以说**：
- ✅ 环境能量、空间布局、氛围营造
- ✅ 心理状态、性格特质、天赋潜能
- ✅ 传统文化、历史智慧、决策思维
- ✅ 养生调理、健康养护、生活方式

### 铁律二：不承诺效果——不保证发财、不保证治病、不保证改运

**禁止承诺**：
- ❌ "我保证你发财"
- ❌ "一定能治好你的病"
- ❌ "这次改运后一定会好"
- ❌ "包准、包灵、包你满意"

**合规表达**：
- ✅ "从传统文化角度，这属于……"
- ✅ "建议你可以尝试……"
- ✅ "在古人看来，这种情况适合……"
- ✅ "仅供参考，最终选择权在你"

### 铁律三：按咨询收费——不是"做法"，是"出方案"

**禁止收费**：
- ❌ 法事费、改运费、消灾费、开光费
- ❌ 灵符费、法器费、超度费

**合规收费**：
- ✅ 课程费、资料费、咨询费
- ✅ 按小时/次计费的咨询服务
- ✅ 知识付费产品的标准定价

**核心理念**：
做干净的生意，赚踏实的钱，睡安稳的觉。

- [ ] 免责声明自动添加提示

### 6.3 AI转译助手

**AI转译助手应具备的能力**：

```python
# 伪代码示例
def translate_term(input_text):
    # 敏感词映射表
    translation_map = {
        "算命": "性格分析",
        "改运": "调整状态",
        "空亡": "能量虚浮",
        "克父": "父子缘分特别",
        "治疗": "调理",
        # ... 更多映射
    }

    # 检测并转译
    for forbidden, allowed in translation_map.items():
        if forbidden in input_text:
            return input_text.replace(forbidden, allowed)

    return input_text
```

### 6.4 AI风险评估

**风险等级定义**：

| 风险等级 | 触发条件 | 建议操作 |

| 🔴 高风险 | 包含医疗承诺、改运收费、重大决策承诺 | 禁止发布，必须修改 |
| 🟡 中风险 | 包含敏感词汇、缺少免责声明 | 强烈建议修改 |
| 🟢 低风险 | 敏感词已转译、免责声明完整 | 可发布，建议人工复核 |

## 🚨 八、分平台专属准则与铁律条款

> 本章节详细规定了抖音、小红书、视频号三大平台的高压电铁律，必须严格遵守

### 📱 8.1 抖音平台 · 专项执行准则与铁律

**平台特性分析**：
算法审核极度灵敏，对"引导互动"、"虚假人设"、"绝对承诺"打击严厉。流量分发依赖于完播率与互动健康度。

| 条款编号 | 铁律内容（必须执行） | 风险场景与应对逻辑 |
|---------|---------------------|-------------------|
| DY-001 | **禁止"评论区接龙算命"**：严禁在评论区引导用户发"出生日期"、"测测我"、"接好运"、"还愿"等具有明确命理指向的词汇。 | **风险**：触发"封建迷信"批量限流或下架。<br>**合规操作**：互动钩子改为"扣'能量'领取自测指南"或"扣'1'听下一期决策干货"。 |
| DY-002 | **直播间背景与人设铁律**：背景禁止出现神像、八卦镜、符箓、超大罗盘；主播着装禁止穿着全真/正一一模一样的法衣、道袍。 | **风险**：被系统判定为"假冒宗教人士"或"传播迷信内容"。<br>**合规操作**：背景使用书架、古琴、文房四宝、国潮字画；着装以新中式、禅服、汉服常服为主。 |
| DY-003 | **连麦咨询铁律**：连麦观众若哭诉病情、求治病、求挽回小三，主播必须立刻打断。 | **风险**：诱导粉丝冲动消费、涉及医疗与情感干预，极易被同行录屏举报。<br>**合规话术**："这位朋友，关于身体健康我们必须遵循医嘱，我不懂医术；关于感情，我们只从'能量匹配度'和'沟通方式优化'的文化视角探讨，不做成败定论。" |
| DY-004 | **"空亡"一词的抖音特规**：抖音对"亡"、"死"、"绝"音视频识别极严。口播时建议用"信号弱"、"能量虚浮"完全替代"空亡"发音，字幕中绝对不可出现"空亡"二字。 | **风险**：直接触发AIGC内容审核红线，视频仅自己可见或账号被评级。 |

**抖音审核重点**：
- 玄学类内容需严格规避敏感词
- 直播需明确声明"非预测"
- 不能出现具体效果的承诺
- 禁止"评论区接龙算命"
- 禁止使用"空亡"、"落空"、"一场空"等高敏感词汇

**推荐人设**：
- "传统文化研究者"
- "时空决策逻辑分享者"
- "古代智慧现代学者"

### 📕 8.2 小红书平台 · 专项执行准则与铁律

**平台特性分析**：
强搜索逻辑，重图文审美，用户对"广告硬广"、"站外导流"极度反感。审核侧重"虚假人设"、"制造容貌/财富焦虑"。

| 条款编号 | 铁律内容（必须执行） | 风险场景与应对逻辑 |
|---------|---------------------|-------------------|
| XHS-001 | **严禁"邪门"式标题党**：笔记封面和标题禁止出现"克夫相自救指南"、"倒霉脸的三个特征"、"这种面相注定穷"。 | **风险**：制造容貌焦虑与社会焦虑，直接违规下架并扣信用分。<br>**合规操作**：标题改为《东方审美观｜让面庞看起来更舒展的3个微调思路》、《古人说的"福气"其实是一种高能量的表情管理》。 |
| XHS-002 | **私信导流铁律**：私信自动回复中，严禁直接出现微信号、手机号、二维码、外链。 | **风险**：账号被禁言、限流、甚至封禁。<br>**合规操作**：引导用户进小红书群聊，在群聊内通过"公告"或"群文件"发布合规的《潜能观察表》获取方式；或使用小红书店铺上架"1元咨询链接"进行交付转化。 |
| XHS-003 | **图文笔记排版铁律**：排版中若使用罗盘、八卦图素材，必须叠加"仅供娱乐"、"传统文化分享"的贴纸水印。 | **风险**：被机器误判为风水号。<br>**合规操作**：每一张图都要有明确的文字水印覆盖核心敏感图案区域，证明是"知识分享"而非"业务揽客"。 |
| XHS-004 | **评论区稽查铁律**：必须安排专人巡查评论区。若有用户在评论区说"大师帮我算算婚姻"，官方号严禁回复"私你排盘了"。 | **风险**：用户在评论区留下的违规痕迹同样会导致笔记限流。<br>**合规回复**："亲爱的，我不做预测哈。只是从传统文化视角分析性格特质，具体选择要看你自己的心意哦~" |

**小红书审核重点**：
- 严禁站外导流（微信号、手机号、外链）
- 封建迷信内容直接下架
- 虚假宣传、制造焦虑会被限流
- 严禁"邪门"式标题党，禁止制造容貌焦虑

**推荐人设**：
- "决策逻辑教练"
- "传统文化学习爱好者"
- "古代智慧研究学者"

**引流合规路径**：
1. 笔记钩子 → 2. 私信自动回复（无联系方式） → 3. 引导进群 → 4. 课程转化

### 📺 8.3 视频号平台 · 专项执行准则与铁律

**平台特性分析**：
基于微信社交强关系链，私域转化最顺畅，但长辈用户多、举报门槛极低、举报后果极重。内容审核偏重"正能量"、"积极价值观"。

| 条款编号 | 铁律内容（必须执行） | 风险场景与应对逻辑 |
|---------|---------------------|-------------------|
| SPH-001 | **朋友圈+直播间"双域合规"铁律**：视频号直播可以引流企业微信，但直播口播和朋友圈文案必须严格同步。 | **风险**：视频号封禁会连带影响微信社交功能（朋友圈、群聊）。<br>**合规操作**：视频号直播话必须与抖音保持同等级别的合规标准，不可因私域就放松警惕。 |
| SPH-002 | **"长辈关怀"铁律**：严禁使用"不这样做你家孩子就废了"、"家里老人身体不好是因为某个摆设"这类针对老人、孩子的恐惧营销。 | **风险**：中老年用户会直接截图向微信官方投诉，引发人工审核严查。<br>**合规操作**：文案核心必须导向"亲子沟通方法"、"家居舒适度提升"。 |
| SPH-003 | **国学/养生课程交付铁律**：视频号橱窗售卖的国学课程、养生食品，详情页必须置顶《非医疗/非承诺告知书》截图。 | **风险**：微信小商店的投诉入口非常便捷，一旦被投诉虚假宣传，资金冻结周期长。<br>**合规操作**：主图第五张必须为白底黑字的免责声明。 |
| SPH-004 | **"群托"管控铁律**：微信群内严禁安排"托"发布"我听了老师的课病好了"、"听了老师的改运赚了50万"等虚假反馈。 | **风险**：群内聊天记录被截屏举报，按《反不正当竞争法》和微信生态规则，属欺诈封号，永不解封。 |

**视频号审核重点**：
- 遵守腾讯平台内容规范
- 避免极端化表述
- 内容导向需积极健康
- 严禁针对老人、孩子的恐惧营销
- 严禁微信群虚假反馈
