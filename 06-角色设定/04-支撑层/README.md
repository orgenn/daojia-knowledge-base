| 角色 | 代号 | 核心价值 |
|------|------|---------|

## 层级定位

支撑层是团队的**运营底座**，负责：

1. **私域运营**：社群运营官管理微信社群，完成深度转化
2. **智能辅助**：AI智脑助理为全团队提供实时智能支持

---

## 角色协作

```
平台运营专员（三大平台）
    ↓ 导入流量
社群运营官（群官）
    ↓ 免费干货建立信任
课程/咨询转化

    （同时）

团队所有成员
    ↓ 提出需求
AI智脑助理（道承智脑）
    ↓ 输出结果
合规检查 / 话术转译 / 内容生成 / 选题规划
```

---

## 社群运营官（群官）

### 核心职责
- 管理微信社群（项目私域核心资产）
- 每日输出免费干货，建立专业信任
- 设计转化漏斗，完成课程/咨询销售
- 处理敏感话题，确保群内合规

### 每日运营模板
| 时间 | 内容类型 | 来源 |
|------|---------|------|
| 早间8:00 | 今日能量播报 | 卜模块日更 |
| 午间12:00 | 知识卡片/短内容 | 五模块之一 |
| 晚间20:00 | 互动话题/投票 | 自创或热点 |
| 每周五 | 课程/活动预告 | 转化导向 |

### 转化漏斗
```
进群
    ↓
领取免费干货（建立初步信任）
    ↓
参与群活动（深度互动）
    ↓
体验低价产品（1元课程/免费咨询）
    ↓
购买核心产品（完整课程/深度咨询）
    ↓
成为老客户（复购+转介绍）
```

## AI智脑助理（道承智脑）

### 四大工作模式

| 模式 | 触发词 | 输出 |
|------|--------|------|
| **合规检查模式** | 检查/审核/这条能发吗 | 合规审核报告（风险等级+问题+修改建议+最终结论） |
| **话术转译模式** | 翻译/转译/怎么说 | 话术转译（原词+风险+替换方案+使用场景+示例） |
| **内容生成模式** | 写/生成/帮我出 | 合规内容（自动规避禁用词+套用模板+附免责声明） |
| **选题规划模式** | 选题/本周内容/帮我排 | 内容周历（标题方向+目标平台+话术角度+合规提示） |

### 内置知识库
- 全套13个角色提示词
- 所有平台合规规则（抖音/小红书/视频号）
- 五术模块话术转换表
- 标准免责声明模板
- 平台内容结构模板

### 使用示例
```
@道承智脑 帮我排下周的内容计划，下周一是立夏
→ [输出下周内容周历，重点突出立夏节气相关内容]

@道承智脑 检查这条脚本能发吗：[脚本内容]
→ [输出合规审核报告，包含修改建议]

@道承智脑 帮我写一条关于清明养生的抖音短视频
→ [输出完整短视频脚本，已含免责声明]

@道承智脑 空亡这个词怎么说
→ [输出话术转译结果]
```

## 共同价值

### 社群运营官的价值
- 将公域流量转化为私域资产
- 通过持续价值输出建立深度信任
- 完成高客单价产品的销售转化

### AI智脑助理的价值
- 提升团队内容生产效率
- 降低合规风险（前置检查）
- 统一话术标准（话术转译）
- 释放人力资源（自动化内容生成）
