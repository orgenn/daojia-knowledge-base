## 层级定位

五术模块专官层是团队的**内容生产引擎**，负责：

1. **现代转译**：将传统山医命相卜转化为当代可理解的内容
2. **模块专精**：每个角色专注一个模块，深度专业化
3. **内容产出**：为三大平台持续提供高质量内容

## 内容生产协作

### 选题分配
内容总监（排局官）根据选题矩阵，将选题分配给对应模块专官。

### 生产流程
```
排局官分配选题
    ↓
对应模块专官（山/医/命/相/卜）接收选题
    ↓
按该模块话术规则生产内容
    ↓
提交排局官初审
    ↓
提交护法官终审
    ↓
平台运营专员发布
```

### 联动选题
- **亲子专题**：命模块（性格分析）+ 医模块（儿童健康）
- **职场专题**：卜模块（决策时机）+ 命模块（职业天赋）
- **家居专题**：相模块（空间设计）+ 山模块（家庭养生）

## 共同原则

所有五术模块专官都必须遵守：

1. **禁用词规避**：不使用算命/占卜/改运/克父/治疗等词汇
2. **去神秘化**：把"天机"变成"规律"，把"神授"变成"模型"
3. **去恐惧化**：不做"不这样做会怎样"的恐吓式表达
4. **不承诺效果**：不保证任何具体结果（赚钱/治病/改命）
5. **附免责声明**：健康类和命理类内容必须加免责声明
