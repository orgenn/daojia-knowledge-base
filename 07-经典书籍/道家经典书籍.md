# 道家经典书籍

## 核心经典（必读）

### 1. 道德经
- **作者**: 老子（李耳）
- **成书**: 春秋时期（约公元前571年-公元前471年）
- **字数**: 约5000字
- **地位**: "万经之王"
- **核心**: 道法自然、无为而治
- **在线阅读**: https://ctext.org/dao-de-jing/zhs

### 2. 清静经
- **全称**: 《太上老君说常清静经》
- **字数**: 约400字
- **地位**: 道教心性修养第一经典
- **核心**: 清静为天下正

### 3. 阴符经
- **全称**: 《黄帝阴符经》
- **字数**: 约300-400字
- **核心**: 天人合一、阴阳消长

### 4. 抱朴子
- **作者**: 葛洪（东晋）
- **结构**: 内篇20卷 + 外篇50卷
- **内容**: 神仙、炼丹、符箓
- **地位**: 确立道教神仙体系

### 5. 黄庭经
- **地位**: 上清派核心经典
- **核心**: 身神养生

## 学习路径建议

### 入门（1-3个月）
1. 道德经 → 清静经 → 阴符经
2. 八字基础（子平真诠入门）
3. 风水基础（葬经入门）

### 进阶（3-6个月）
1. 内丹：悟真篇 → 参同契
2. 符箓：道藏符箓部
3. 奇门：烟波钓叟歌

### 专修（6个月+）
根据兴趣选择：八字/风水/奇门/相术/内丹
