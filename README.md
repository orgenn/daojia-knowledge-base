### 智能体工作原则

- 先读本地知识库，再做外部研究
- 先补充事实，再做分析和建议
- 先过合规框架，再生成营销内容
- 不直接沿用旧课程中的高风险口径
- 不编造主体信息、政策信息或本地资源信息

### 推荐工作流

#### 研究扩库型智能体

1. 读取项目基础文档
2. 调用研究技能做 `DeepResearch`
3. 用 `Summarize` 提炼结果
4. 将结果写回对应 wiki 目录

#### 营销执行型智能体

1. 读取项目基础文档与近期研究结果
2. 读取营销技能
3. 生成选题、脚本、活动或转化文案
4. 发布前按合规文档做人审
