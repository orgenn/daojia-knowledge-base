## 主IP：云笈仙人

### 人物设定

| 属性 | 描述 |
|------|------|
| **身份** | 云笈七签的守护者 |
| **外貌** | 白衣玄衫，手持竹简，身边环绕淡淡云雾 |
| **性格** | 话不多，但说一句就是一句；不急不躁，自有节奏 |
| **能力** | 能从"云笈"中取出对应签的智慧 |
| **使命** | 引导现代人找到自己的修行路，不盲从、不迷信 |
| **现代感** | 偶尔拿出手机，用"时"来解释"道" |

### 设计理念

- 不是传统道士形象，是"现代修行者"
- 有温度的知识传播者，不是高高在上的大师
- 用现代语言解释古代智慧

### 应用场景

| 场景 | 应用形式 | 商业价值 |
|------|---------|---------|
| **短视频** | 云笈仙人讲解七签 | 知识IP人格化 |
| **直播** | 仙人出镜答疑互动 | 建立信任+情感连接 |
| **表情包** | 云笈仙人衍生表情 | 社交传播+用户共创 |
| **NFT/数字藏品** | 云笈书箱3D模型 | 科技感+收藏价值 |
| **线下快闪** | 仙人装置+AR互动 | 沉浸式体验+打卡传播 |

## 七签使者IP矩阵

### 使者设计原则

- ✅ 现代化：不是传统道士形象，是"现代修行者"
- ✅ 可识别：每个使者有独特视觉符号
- ✅ 可延展：适合做表情包、周边、联名

### 使者完整档案

| 使者 | 代表签 | 象征物 | 核心特质 | 代表色 | 喜欢表达 | 现代场景 |
|------|--------|--------|---------|--------|----------|---------|
| **藏经子** | 第一签·藏经 | 竹简、书卷 | 博学、温柔 | 🟡 金黄 | "知识是地图，不是终点" | 知识学习、读书成长 |
| **养性子** | 第二签·养性 | 铜镜（照心）| 敏锐、善问 | 🟠 浅紫 | "照见自己，才能改变" | 心理成长、自我觉察 |
| **炼命子** | 第三签·炼命 | 丹炉（炼养）| 务实、坚持 | 🟢 森绿 | "身体是修行的道场" | 健康养生、身体调理 |
| **知时子** | 第四签·知时 | 罗盘（择时）| 洞察、果敢 | 🔵 深蓝 | "时机到了，自然水到渠成" | 决策时机、职业规划 |
| **齐家子** | 第五签·齐家 | 结绳（连结）| 温暖、包容 | 🟧 橙红 | "关系是修行的道场" | 亲子关系、家庭和谐 |
| **处世子** | 第六签·处世 | 流水（上善）| 灵动、从容 | 🔷 青蓝 | "上善若水，利万物" | 职场智慧、压力管理 |
| **归真子** | 第七签·归真 | 星空（天人）| 超然、释然 | 🟣 银白 | "天人合一，当下即永恒" | 生命意义、精神追求 |

## IP运营策略

### 阶段1：IP人格建立（0-3个月）

**核心动作**：
- 发布"云笈仙人"首支短视频（自我介绍）
- 七签使者逐一登场（每周1个使者）
- 建立IP世界观（云笈书箱的由来、使命）

**内容示例**：
```
【云笈仙人首秀视频】

[0-5秒] 画面：云雾散开，白衣身影显现
"人们说，道家很远。其实它一直在你身边……"

[5-25秒] 画面：打开云笈书箱，七签光芒流转
"这是我守护千年的云笈，里面藏着七个生活智慧。"
"第一签·藏经：知识不是终点，是地图。"
"第二签·养性：照见自己，才能改变……"

[25-45秒] 画面：云笈书箱合上，递向镜头
"你想找到你的守护签吗？"
"评论区告诉我你最想修炼什么，我来安排。"

[45-60秒] 画面：云笈仙人微笑消失
"本内容基于《云笈七签》传统文化研究。"
```

### 阶段2：IP关系深化（3-6个月）

**核心动作**：
- 七签使者独立内容（每个使者1条短视频/周）
- 用户互动（"哪位使者最像你？"投票活动）
- 使者联动（七签合体故事）

### 阶段3：IP生态延展（6-12个月）

**核心动作**：
- 用户共创（七签使者二创、同人故事）
- 跨界联动（与其他IP破圈）
- 线下实体（快闪店、巡展）
