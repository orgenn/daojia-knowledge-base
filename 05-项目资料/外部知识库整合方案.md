## 三、整合步骤

### 3.1 第一步：导入GBrain

```bash
# 导入灏泽知识库
~/.bun/bin/gbrain import ~/文档/haoze/正文/ --no-embed

# 导入玄门知识库
~/.bun/bin/gbrain import ~/文档/xuanmen/wiki/ --no-embed

# 生成嵌入
~/.bun/bin/gbrain embed --stale
```

### 3.2 第二步：创建符号链接

```bash
# 在道家知识库中创建外部知识库链接
ln -s ~/文档/haoze ~/文档/道家知识库01/基础RAW/灏泽知识库
ln -s ~/文档/xuanmen ~/文档/道家知识库01/基础RAW/玄门知识库
```

### 3.3 第三步：同步到OpenCode

```bash
# 更新OpenCode同步
~/.opencode/sync-skills.sh
```

### 3.4 第四步：创建选题库

基于灏泽知识库的高流量选题，创建新媒体选题库：

```bash
# 创建选题库目录
mkdir -p ~/文档/道家知识库01/02-商业运营/03-内容与传播/选题库

# 从灏泽知识库提取选题
python3 ~/文档/haoze/classify_and_index.py
```

- 发布需经过合规检查
