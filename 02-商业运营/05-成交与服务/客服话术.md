# 客服话术

> 售前·售中·售后

---

## 一、售前话术

### 首次接触

| 场景 | 话术 |
|------|------|
| 打招呼 | 您好！我是道小童，很高兴认识您~ |
| 了解需求 | 您是想了解哪方面呢？养生/命理/风水？ |
| 介绍服务 | 我们有免费的入门资料，您可以先看看~ |

### 价值塑造

| 场景 | 话术 |
|------|------|
| 介绍课程 | 这个课程是老师多年心得，很实用~ |
| 介绍咨询 | 老师经验丰富，帮助很多人解决了问题~ |

---

## 二、售中话术

### 成交促进

| 场景 | 话术 |
|------|------|
| 优惠 | 今天报名有优惠哦~ |
| 限时 | 名额有限，尽快锁定~ |
| 保障 | 不满意可以退款的~ |

### 异议处理

| 问题 | 回应 |
|------|------|
| 太贵了 | 价值在这里，效果看得见~ |
| 再考虑 | 好的，有问题随时问我~ |
| 效果呢 | 之前学员反馈都不错~ |

---

## 三、售后话术

### 交付

| 场景 | 话术 |
|------|------|
| 开课通知 | 课程已开通，您可以开始学习啦~ |
| 资料发送 | 资料已发送，请查收~ |

### 跟进

| 场景 | 话术 |
|------|------|
| 学习反馈 | 学完感觉怎么样？有问题随时问~ |
| 复购推荐 | 我们还有进阶课程，您要不要了解~ |
