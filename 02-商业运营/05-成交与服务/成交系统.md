## 一、转化漏斗

### 漏斗模型

| 阶段 | 转化率 | 动作 |
|------|--------|------|
| 曝光→点击 | 3% | 标题/封面 |
| 点击→添加 | 10% | 主页/内容 |
| 添加→私域 | 50% | 话术 |
| 私域→付费 | 3% | 产品/转化 |

## 二、转化话术

### 需求挖掘

```
您是想了解哪方面呢？
- 职业发展
- 情感婚姻
- 健康养生
- 家庭关系
```

### 痛点放大

```
很多人都有类似的困惑：
[痛点描述]

如果不解决，会[后果]

其实可以通过[方案]改善~
```
