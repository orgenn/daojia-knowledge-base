# 用户画像

> 徐州云笈文化当前阶段优先服务的人群，不追求“大而全”，先抓高匹配、高信任、高复购的人群

---

## 一、画像使用原则

当前阶段的用户画像只服务三个目的：

1. 决定内容该讲给谁听
2. 决定先卖什么产品
3. 决定私域里先重点跟进谁

因此，本页不做泛泛人群描述，优先锁定可转化、可长期服务的人群。

---

## 二、C 端核心人群

### 画像 A：28-35 岁的城市女性白领

| 维度 | 描述 |
|------|------|
| **基本特征** | 一二线或新一线工作，审美在线，愿意为成长和情绪稳定付费 |
| **核心痛点** | 焦虑、内耗、睡不好、生活秩序混乱、关系消耗大 |
| **真实需求** | 想要更稳的情绪、更顺的日常节奏、更柔和但不玄的“方法感” |
| **高频表达** | “我就是太累了”“我脑子停不下来”“我想找一点让我静下来的东西” |
| **更容易被什么打动** | 节气养生、情绪稳定、家居秩序、女性日常调理、慢生活审美 |
| **适合产品** | 9.9 资料包、99 微课、7天打卡营、节气主题社群内容 |
| **主要渠道** | 小红书、视频号、朋友圈 |
| **价格敏感度** | 对 9.9-199 元接受度高，对 399+ 需要先建立信任 |

### 画像 B：35-45 岁的家庭管理型女性

| 维度 | 描述 |
|------|------|
| **基本特征** | 已婚已育概率高，是家庭情绪和消费的重要决策者 |
| **核心痛点** | 家庭关系紧绷、孩子习惯难带、自己精力不足、家里乱、长期疲惫 |
| **真实需求** | 想找到能改善家庭氛围、孩子作息、自己状态的方法 |
| **高频表达** | “家里总是乱”“孩子不听”“我没有自己的时间”“最近身体很虚” |
| **更容易被什么打动** | 家庭秩序、亲子沟通、节气生活、养生饮食、空间整理与氛围 |
| **适合产品** | 家庭主题资料包、21天关系/养生小营、线下沙龙、周末体验活动 |
| **主要渠道** | 视频号、私域、熟人转介绍 |
| **价格敏感度** | 对 99-399 元产品接受较好，对线下活动和家庭型产品更容易付费 |

### 画像 C：40-55 岁的健康焦虑型中产

| 维度 | 描述 |
|------|------|
| **基本特征** | 有一定消费能力，更关心身体、家庭和人生节奏 |
| **核心痛点** | 睡眠差、精力差、压力大、对未来有隐性焦虑 |
| **真实需求** | 需要一种不神秘、不激进、能长期坚持的生活方式方案 |
| **高频表达** | “最近睡眠不好”“感觉身体在走下坡路”“想调整但不知道从哪开始” |
| **更容易被什么打动** | 养生、节气、静心、日常作息、饮食与居家环境 |
| **适合产品** | 养生主题课、线下沙龙、长期会员、年度陪伴类服务 |
| **主要渠道** | 视频号、私域、线下活动 |
| **价格敏感度** | 对低价不敏感，更在意信任、口碑和交付感受 |

## 三、B 端可测试人群

### 画像 D：本地中小企业主 / 空间主理人

| 维度 | 描述 |
|------|------|
| **核心需求** | 团队状态、空间氛围、品牌活动、员工关系、企业家自我调节 |
| **切入话题** | 决策节奏、团队状态、空间秩序、主理人能量管理 |
| **适合产品** | 小型沙龙、闭门分享会、企业定制活动、空间合作活动 |
| **成交难点** | 需要真实案例、可信身份、清晰产品边界 |

当前建议：

- 先作为合作资源和线下活动资源经营
- 不作为第一阶段主力收入来源
- 等 C 端案例和内容口碑积累后再放大

---

## 四、优先级排序

### 当前阶段优先顺序

1. `画像 A`：最适合做内容起盘和种子用户沉淀
2. `画像 B`：最适合做私域转化和线下活动
3. `画像 C`：最适合做长期服务和高信任产品
4. `画像 D`：作为合作与后期企业服务储备

---

## 五、用户决策触发点

### 最容易发生转化的触发时刻

| 场景 | 用户当下状态 | 适合推什么 |
|------|------|------|
| **换季/节气变化** | 想调整身体和生活节奏 | 节气资料包、养生打卡营 |
| **家庭关系紧张** | 想改善家里氛围 | 家庭关系主题内容、沙龙、社群 |
| **工作压力大** | 想找到稳定自己的方法 | 情绪稳定内容、短课、直播答疑 |
| **家居混乱** | 想改善空间和心情 | 空间整理、美学、秩序感内容 |
| **睡眠与疲惫问题** | 想先恢复基本状态 | 作息、饮食、静心、轻养生产品 |

## 六、用户最常见顾虑

### 反对意见库

| 顾虑 | 本质问题 | 应对方式 |
|------|------|------|
| **怕太玄** | 不信任、怕踩坑 | 强调文化传播、生活方式、可操作方法 |
| **怕没用** | 过去踩过坑 | 用案例、过程感、日常小改变替代夸张承诺 |
| **怕太贵** | 没看见价值 | 用低门槛体验品建立第一次信任 |
| **怕坚持不了** | 自控力不足 | 打卡、陪伴、社群、轻量任务设计 |
| **怕被推销** | 关系防御强 | 先给内容价值，再做自然转化 |

## 七、内容与产品匹配表

| 画像 | 首选内容 | 首选产品 | 首选转化动作 |

| **B 家庭管理型女性** | 家庭关系、孩子习惯、家庭节奏 | 99-399 主题营、线下沙龙 | 进群、咨询、报名活动 |

## 八、用户访谈要点

后续做用户访谈时，建议至少问清：

1. 你最近最想解决的一个问题是什么
2. 你现在最累的是身体、关系还是工作
3. 你过去尝试过哪些方法，为什么没坚持
4. 你更愿意从内容、课程、社群还是线下活动开始
5. 你能接受的首次付费区间是多少

---

## 九、不优先服务的人群

以下人群不建议作为前期主力目标：

- 强烈追求“预测”“改运”“法事”类结果的人
- 只想白嫖、对长期价值极弱的人
- 只看最低价、不看服务体验的人
- 对项目定位长期不认同的人
