# 内容规划

> 以“内容起盘 + 私域承接 + 产品转化”为目标的实操版内容矩阵

---

## 一、内容目标

当前阶段内容不追求“大而全”，只服务四个目标：

1. 让目标用户停下来
2. 让用户觉得“这个内容和我有关”
3. 让用户愿意进入私域
4. 让私域里的人逐步转化为付费用户

---

## 二、内容核心定位

### 统一表达方向

- 把传统文化翻译成现代生活方法
- 把抽象道理落到身体、家庭、空间、关系、节奏
- 不讲玄，不断语，不恐吓，不夸张

### 一句话内容口径

`讲得清、做得到、能坚持、不过线`

---

## 三、四大内容支柱

### 支柱 1：养生与日常秩序

适合话题：

- 节气生活
- 作息与睡眠
- 饮食与体感
- 日常小功法
- 情绪稳定与身体恢复

### 支柱 2：家庭与关系

适合话题：

- 家庭氛围
- 亲子沟通
- 伴侣相处
- 家中秩序与情绪
- 如何让家“更稳”

### 支柱 3：职场与人生节奏

适合话题：

- 压力管理
- 决策节奏
- 内耗与专注
- 适合什么时候慢下来
- 如何在乱局里恢复秩序感

### 支柱 4：空间与生活美学

适合话题：

- 空间整理
- 家居氛围
- 房间对情绪的影响
- 传统审美和当代生活结合
- 居家“充电感”

---

## 四、内容漏斗

| 层级 | 内容目的 | 内容形式 | CTA |
|------|------|------|------|
| **流量层** | 抢注意力 | 反常识、痛点、节气钩子、短观点 | 收藏 / 评论 / 关注 |
| **信任层** | 建立专业感 | 方法拆解、案例、清单、问答 | 私信 / 进群 / 直播预约 |
| **转化层** | 完成咨询或报名 | 产品说明、结果路径、用户故事 | 咨询 / 报名 / 购买 |
| **留存层** | 强化复购与陪伴 | 打卡、复盘、社群互动、活动记录 | 复购 / 转介绍 / 续费 |

## 五、平台分工

### 抖音

定位：短视频流量入口

- 内容占比：`流量层 70% + 信任层 30%`
- 推荐形式：30-60 秒口播、情境式短视频、直播切片
- 推荐 CTA：评论关键词、私信资料、进直播间

### 小红书

定位：种草 + 搜索 + 高质量私域线索

- 内容占比：`信任层 60% + 流量层 20% + 转化层 20%`
- 推荐形式：图文笔记、轮播卡、方法清单、案例拆解
- 推荐 CTA：收藏、私信、进群、领取清单

### 视频号

定位：信任与转化中枢

- 内容占比：`信任层 50% + 转化层 30% + 留存层 20%`
- 推荐形式：短视频、直播、活动记录、主理人口播
- 推荐 CTA：进群、咨询、报名直播或活动

### 私域

定位：沉淀、培育、成交、复购

- 内容占比：`留存层 40% + 信任层 30% + 转化层 30%`
- 推荐形式：早安内容、问答、案例、直播预告、活动通知

## 六、内容栏目设计

### 栏目 A：节气生活提醒

定位：稳定输出，适合高频更新

- 内容例子：立夏后最该先调整的 3 个习惯
- 适合平台：抖音、小红书、私域

### 栏目 B：家的秩序感

定位：强共鸣，适合转化女性与家庭人群

- 内容例子：为什么家里一乱，人就更容易发脾气
- 适合平台：小红书、视频号、私域

### 栏目 C：职场减耗

定位：连接白领用户，建立“生活方法”认知

- 内容例子：越忙的时候，越要先把这件小事做对
- 适合平台：抖音、视频号

### 栏目 D：传统智慧现代翻译

定位：品牌辨识度栏目

- 内容例子：把“无为”翻译成今天能做到的一件事
- 适合平台：抖音、视频号、小红书

### 栏目 E：用户真实问题拆解

定位：最接近成交

- 内容例子：最近总失眠、总烦躁的人，先别急着“补”
- 适合平台：小红书、视频号、私域

## 七、每周内容配比

### 建议周节奏

| 类型 | 数量 | 说明 |
|------|------|------|
| **流量型内容** | 3 | 负责曝光和停留 |
| **信任型内容** | 2 | 负责收藏、私信、关注 |
| **转化型内容** | 1 | 负责直播、活动、产品承接 |
| **私域留存内容** | 3-5 | 负责陪伴、转化、复购 |

### 推荐内容比例

`流量 : 信任 : 转化 = 5 : 3 : 1`

---

## 八、选题标准

### 一条内容值得做，至少要满足以下一项

- 用户会明显共鸣
- 能直接解决一个小问题
- 能让用户愿意收藏或转发
- 能把人自然带入私域
- 能为某个产品或活动预热

### 尽量避免

- 太空泛的大道理
- 太重的术语堆砌
- 太像“知识背诵”的表达
- 太接近高风险玄学口径的题材

---

## 九、内容生产模板

### 短视频模板

1. 3 秒钩子
2. 点出一个问题
3. 讲一个核心观点
4. 给一个具体动作
5. 给 CTA

### 图文模板

1. 封面标题
2. 痛点共鸣
3. 3-5 个方法点
4. 一个适合收藏的总结
5. 私信 / 进群 CTA

### 直播预热模板

1. 主题
2. 适合谁
3. 能解决什么问题
4. 不适合谁
5. 直播时间和入口
